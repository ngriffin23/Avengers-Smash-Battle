Avengers Smash Battle
===================

This is a browser game of rock paper scissors with a twist. 

Rules
--------
Select a character and battle the computer. Whoever lose gets hit.

- Thanos beats Captain America
- Hulk beats Thanos
- Captain beats Hulk
